# Portafolio Azure Empresarial

Portafolio de proyectos practicos en Azure organizados por progresion de certificacion,
desde conceptos fundamentales (AZ-900) hasta arquitecturas de seguridad avanzada (AZ-500).

## Nivel de las competencias demostradas

Estos proyectos exceden el alcance conceptual de AZ-900. Cubren competencias practicas
alineadas a AZ-104 (Administrator), AZ-204 (Developer) y AZ-500 (Security Engineer).
