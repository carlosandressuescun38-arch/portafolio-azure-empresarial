# Portafolio Azure — Fundamentos (AZ-900)

Portafolio de proyectos practicos alineados a la certificacion Microsoft Azure
Fundamentals (AZ-900), desarrollados 100% en Azure Portal sobre una cuenta
Azure for Students.

## Proyectos

| Proyecto | Concepto demostrado | Estado |
|---|---|---|
| A1-primer-recurso | Resource Groups: organizacion logica de recursos | Completo |
| A2-almacenamiento-blob | Azure Storage: tipos de almacenamiento (Blob) | Completo |
| A3-maquina-virtual-basica | IaaS vs PaaS: modelos de servicio en la nube | Completo |
| A4-presupuesto-alertas | Cost Management: gobernanza de costos (OpEx) | Completo |
| A5-explorar-rbac-politicas | RBAC y Azure Policy: gobernanza y control de acceso | Completo |

## Metodologia
Cada proyecto incluye:
- Concepto teorico de AZ-900 aplicado en la practica
- Accion realizada paso a paso via Azure Portal
- Evidencia visual (capturas de pantalla) en su carpeta `/evidence`
- Documentacion de decisiones y observaciones tecnicas

## Suscripcion utilizada
Azure for Students (Universidad de los Andes). Ver `suscripciones.md` para detalle.
